# Video Recommendation App

## Description
Application de partage de vidéos avec un système de recommandation basé sur le temps de visionnage et une sélection obligatoire de niche lors de l’upload.

## Design Thinking
- Empathize : Les utilisateurs veulent du contenu pertinent
- Define : Difficulté à trouver du contenu adapté
- Ideate : Recommandation + niche obligatoire
- Prototype : Interface avec feed personnalisé
- Test : Analyse du comportement utilisateur

## Méthodologie
Méthode Agile Scrum avec gestion via Trello.

## Fonctionnalités
- Création de compte
- Connexion
- Upload vidéo avec niche
- Recommandation personnalisée
- Like et commentaire

## Sprint 1 (2 semaines)
- Créer un compte
- Se connecter
- Regarder des vidéos
- Uploader une vidéo
